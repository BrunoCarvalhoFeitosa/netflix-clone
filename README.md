## Sobre o projeto

![netflixFilmsScreen](https://user-images.githubusercontent.com/46093815/133950288-20d232a9-33bb-44b7-adf3-93f3b8483220.png)


Este app é um "clone" do app Netflix, tendo a listagem de filmes, seriados e documentários em alta na plataforma de streaming, possuindo página para cadastro e login que utilizam 
o SQLite baseado em typeorm para gerenciar migrations da base de dados.

### Feito com

* [React.js](https://pt-br.reactjs.org/)
* [Styled-components](https://styled-components.com/)
* [Node.js](https://nodejs.org/en/)
* [SQLite](https://www.sqlite.org/index.html)

### Hospedagem

O app ainda não foi hospedado em nenhuma plataforma, porém você poderá cloná-lo. Basta acompoanhar o próximo tópico.

## Iniciando o projeto

Primeiramente será necessário clonar este projeto em (https://github.com/BrunoCarvalhoFeitosa/NetflixClone.git), após o download será necessário abrir este projeto no seu
editor e no terminal digitar npm install ou yarn install, posteriormente é só executar o script yarn dev ou npm run dev.

* Abra seu editor e instale as dependêcias utilizando o npm install ou yarn install.
* Dentro do folder frontend execute o comando npm run start ou yarn start e dentro do folder backend execute o comando npm run dev ou yarn dev.
* O app abrirá em seu navegador e o servidor com o banco de dados irá estar funcionando por baixo dos panos.
* Para visualizar os dados cadastrados no SQLite, recomendo usar o software Beekeeper Studio (https://www.beekeeperstudio.io/).

### Pré-requisitos

* npm
  ```sh
  npm install npm@latest -g
  ```
  
### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/BrunoCarvalhoFeitosa/NetflixClone.git
   ```
2. Instale os pacotes NPM
   ```sh
   npm install ou yarn install
   ```
   
3. Inicie o front-end
   ```sh
   npm run start ou yarn start
   ```
   
4. Inicie o back-end
   ```sh
   npm run dev ou yarn dev
   ```
