# Sobre o app

Este é um clone de algumas das interfaces do app Netflix.

## Tecnologias
No backend foi utilizado Node.js e SQLite para manutenibilidade do fluxo de cadastro e login, já no frontend foi utilizado React.js para realizar as interações entre o usuário e o servidor.

Para visualizar a aplicação:

### `yarn start`
Dentro do diretório frontend basta executar no terminal o comando yarn start que a aplicação irá se iniciar.

### `yarn dev`
Dentro do diretório backend basta executar no terminal o comando yarn dev que o servidor irá iniciar.
